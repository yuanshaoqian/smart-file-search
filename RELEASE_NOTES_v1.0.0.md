# Smart File Search v1.0.0 更新说明

### ✅ 已修复的问题

1. **启动慢** - 添加启动画面，显示加载进度
2. **界面卡顿** - 使用多线程处理索引和AI操作

## 💡 使用流程

### 首次使用（无AI）

```
1. 下载 SmartFileSearch.exe (30MB)
2. 双击运行
3. 启动画面显示加载进度
4. 界面立即显示
5. 开始搜索文件（无AI功能）
```

### 启用AI功能

```
1. 在设置中启用AI
2. 点击"下载AI模型"
3. 等待下载（1.7GB，约5-10分钟）
4. 下载完成后，AI功能自动启用
5. 可以使用自然语言搜索
```

## ⚠️ 注意事项

1. **AI模型下载**
   - 仅在使用AI功能时下载
   - 下载后保存在 `data/models/` 目录
   - 可手动删除以节省空间

2. **系统要求**
   - Python 3.10+（已打包，无需安装）
   - 4GB RAM（无AI）/ 8GB RAM（有AI）
   - 100MB 磁盘空间（无AI）/ 2GB（有AI）

3. **首次运行**
   - 启动画面显示2-3秒
   - 界面立即显示
   - 可立即使用文件搜索功能
